Alternance Perfectys Télécom

Présentation de l'Alternance

Au sein de l'entreprise Perfectys Télécom, j'ai occupé un poste polyvalent alliant développement web, référencement (SEO) et administration système. Cette expérience m'a permis de moderniser les outils de l'entreprise et de restructurer le système de supervision réseau.

Développement Web & Refonte

Modernisation du site vitrine

J'ai entièrement restructuré le projet web de l'entreprise. L'ancienne architecture a été remplacée pour gagner en maintenabilité :

  • Refactorisation du code JS : regroupement des éléments communs (header, footer) dans un script unique.
  • Migration vers WordPress avec création d'un thème sur-mesure et intégration via Elementor.
  • Mise en place d'une stack de plugins essentiels : Rank Math (SEO), Wordfence (Sécurité), UpdraftPlus (Sauvegardes) et LiteSpeed Cache (Performance).

Portail Captif

Refonte complète du design et du code (HTML/CSS/JS) du portail captif utilisé pour le Wi-Fi invité.

Stratégie SEO & Marketing

Un travail approfondi a été réalisé pour améliorer la visibilité locale ("Télécom Alès") et la crédibilité en ligne :

  • Intégration de données structurées (JSON-LD LocalBusiness) et optimisation des méta-données.
  • Mise à jour de la fiche Google Maps et gestion des avis via le plugin Trustindex.
  • Création de scénarios d'automatisation (via n8n) pour les campagnes d'avis clients.

Infrastructure & Supervision

Migration du Monitoring

J'ai mené la migration du système de supervision historique (Xymon) vers une solution moderne : Uptime Kuma.

  • Recherche et test de solutions open-source.
  • Développement d'un script Python (Webhook) pour l'envoi d'alertes SMS en cas de coupure > 30 min.

Systèmes & Réseaux

  • Déploiement de Nextcloud sur Proxmox via Docker (configuration VLAN et IP statique).
  • Configuration et déploiement d'une dizaine de téléphones VoIP.
